# Node.js TS3 ServerQuery Client

A TS3 ServerQuery interface written in TypeScript.

## Setup/Requirements
TypeScript v3.4+ and Node.js v10.6+.

Start using `npm start`

## Plugins
- [IdleCheck](https://github.com/weedz/node-ts3_idlecheck)
